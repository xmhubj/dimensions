### Naming Schema

- UPPERCASE
  - Constants
  
- TitleCase
  - Classes
  - Exceptions
  
- camelCase
  - GUI functions and methods
  
- lowercase and lowercase_with_underscore
  - everything else
  
- Use an uppercase letter for the first letter of custom module filenames
